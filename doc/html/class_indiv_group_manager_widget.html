Detailed Description

Individual Group Manager Widget.

A widget for managing a Group object, allows user to display and edit properties of a Group. It has methods to rename a Group; display and update the Lights in the Group and update the Group's all containing Lights' state (on/off, Brightness, Hue, Saturation, Transition Time for switching states).
